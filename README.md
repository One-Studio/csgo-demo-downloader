# CSGO Demo Downloader

CSGO Demo 下载器

## 使用方法

- 打开steam
- 关闭csgo
- 复制游戏 gotv demo 的分享链接，粘贴到文本框中，点击解析按钮
- 等待几秒，会打开浏览器并下载

## 下载
